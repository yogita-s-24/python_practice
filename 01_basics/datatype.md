# DataTypes in Python

## 1. Number -

**Example :**
> 1234, 3.1453, 0b111, Decimal(), fractional 

## 2. String - 

**Example :**
> 'span', "Bombee", b`a\x

## 3. List -

**Example :**
> [1, [2, 'three'], 4.5], list(range(10))

## 4. Tuple - 

**Example :**
> (1, 'spam', 4, 'U'), tuple('spam'), namestuple

## 5. Dictionary - 

**Example :**
> {'food': 'spam', 'taste' : 'yum'}, dic(hours=10)

## Set -

**Example :**
> set('abc'), {'a','b','c'}

## File - 

**Example :**
> open('egg.txt'), open(r'C:\ham.bin', 'wb')

## Boolean -

**Example :**
> True, false
